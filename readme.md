## Overview

The research proposes a deep learning based system for detecting Alzheimer's disease using MRI brain scans.

Models evaluated:

- MobileNetV2
- ResNet50
- InceptionV3

An **augmented attention mechanism** was integrated to improve feature extraction and classification performance.

Dataset used:

- ADNI (Alzheimer's Disease Neuroimaging Initiative)

Classification classes:

- Non Demented
- Very Mild Demented
- Mild Demented
- Moderate Demented
